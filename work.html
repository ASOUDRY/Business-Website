My Flix App

My Code

The Live Application

The MyFlix App is a movie list app that lets you choose from a small list of movies, and filter them by name, genre, and director. it also lets you add and remove favorites. a two part that applies both front and back end technology. It utilizes a custom backend server which hosts a list of movies, directors, genres, and users. These are access in the front end via axios. The front end of the app utilizes react, bootstrap, and redux to make cards based off the information fetched by axios, as well as push information to the backend when logging in amnd registering. Finally this app is primarily styled utilizing tachyon css, with small amounts of css written by me.
